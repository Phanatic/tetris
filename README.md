Tetris
======

Tetris while you ops

## Deploy

Play tetris in your terminal - in color - on Stackato!

	stackato push -n

## How do I play?

![tetris](https://raw.github.com/mafintosh/tetris/master/tetris.png)

Simply open a terminal and run

	$ stackato ssh "node tetris"
